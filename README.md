# React States and Life Cycle Methods Tutorial

 React class based components states & Life cycle methods

# Setup

- clone to computer using:

```
git clone https://github.com/birukkebede11/States-LifeCycleMethods.git
```

- run npm install from the root directory.

```
cd States-LifeCycleMethods
npm install
```

- Finally, to start & run the project

```
npm start
```
